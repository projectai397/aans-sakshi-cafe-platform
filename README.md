# AANS Sakshi Cafe Platform

This is the official repository for the AANS Sakshi Cafe Platform, a comprehensive, AI-powered restaurant management and delivery system.

---

## üöÄ Quick Start

To get the platform running locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/aans-sakshi-cafe.git
    cd aans-sakshi-cafe
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Set up your environment:**
    ```bash
    cp .env.example .env
    ```
    Then, fill in the required values in the `.env` file.

4.  **Start the development server:**
    ```bash
    pnpm dev
    ```

The application will be available at [http://localhost:3000](http://localhost:3000).
